Detailed Description

This program sets up two tasks to run two different DC motors with encoders, 4 inter-task shared variables, and a queue.

Upon the microcontroller's reset, the program will wait for an pair of input, Kps and position setpoints. A pair of encoders, motor drivers, and proportional contollers are then set up and controlled as separate cotasks with these input values. These cotasks are run without blocking each other and the responses are printed out.
